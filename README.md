# ZWlang

zwlang is a programming language that uses `U+200B` as code.  
if brainfuck already felt extreme, this is basically the human limit.

## how to use it

steps:

1. write your program using zero width spaces (good luck)
2. run it with the interpreter

normal mode (invisible):

```bash
python zwlang.py program.zw
```
cheat mod (visible):
```bash
python zwlang.py -cheat program.zw
```
